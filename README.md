# مقهى سُيوف · SWORDS CAFE — موقع تعريفي

موقع تسويقي عربي (RTL) لمقهى قهوة مختصة في حائل.

## معلومات النشاط
- **الاسم:** مقهى سُيوف · SWORDS CAFE
- **التصنيف:** قهوة مختصة / مقهى
- **المدينة:** حائل — المنتزه الغربي، طريق الملك فهد
- **التقييم:** 4.3 على خرائط قوقل (597 مراجعة)
- **الهاتف:** 0556462775 · واتساب: 966556462775

## سبب الاختيار
مقهى بتقييم قوي (4.3 من 597 مراجعة) وهوية بصرية واضحة (سُيوف = SWORDS) وصور حقيقية ممتازة للقهوة والأجواء، لكنه بلا موقع إلكتروني — فرصة مثالية لموقع تعريفي راقٍ يجمع القائمة والموقع والتواصل.

## فكرة التصميم
هوية "قهوة مختصة داكنة وجريئة": خلفية إسبريسو/فحمية (#1A1715) مع لمسة فضّية فولاذية (#9BA3A8) وكهرماني دافئ (#C68A3E) وكريمي. موتيف هندسي مستوحى من السيوف (خطوط/زوايا قطرية خفيفة) مستخدم بذوق لا بشكل كرتوني. خط عرض ثقيل (Cairo/Tajawal) مع لاتيني نظيف (Oswald) لكلمة SWORDS.

## التقنيات
- HTML / CSS / JavaScript خام (Vanilla) — بدون أي إطار أو مرحلة بناء.
- خطوط قوقل عبر `<link>` مع preconnect.
- IntersectionObserver للكشف عند التمرير، Lightbox خفيف، Toast، نموذج → واتساب + localStorage.
- JSON-LD نوع `CafeOrCoffeeShop` مع التقييم 4.3/597.

## الصور
صور حقيقية للمقهى من خرائط قوقل (ملف المشروع) في `assets/img/`. تم اختيار أفضلها للهيرو والمعرض والأقسام.

## التشغيل محليًا
```bash
cd swords-cafe
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

## حالة الـ backend
غير مربوط. نموذج الطلب يجهّز رسالة واتساب جاهزة (`wa.me`) ويحفظ نسخة في `localStorage` للعرض التجريبي فقط — لا يوجد خادم أو قاعدة بيانات.
